# Day 10 - Keto Week: 7-Day Keto Diet Suggestion

## Overview
This Python project generates a **1-week keto diet plan** with breakfast, lunch, and dinner suggestions for each day.

## Project Structure
- `diet.py` - Contains meal data and functions to generate a weekly plan.
- `day_10.py` - Clear entry point for the program.
- `requirements.txt` - Python dependencies.
- `Dockerfile` - To containerize the app.
- `.dockerignore` - Files to ignore during Docker build.

## Usage

### Run locally
```bash
python day_10.py
